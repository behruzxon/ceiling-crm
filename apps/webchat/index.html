Configure the widget and click Apply to reload it with the new settings. The chat button appears at the bottom-right corner of this page.

Embed on Your Website

Add these lines before </body> on any page: 

<link rel="stylesheet" href="https://cdn.example.com/widget.css">
<script src="https://cdn.example.com/widget.js"></script>
<script>
  window.SystemXChat.init({
    tenantSlug:     "your-company",
    apiBaseUrl:     "https://api.example.com",
    title:          "AI Yordamchi",
    primaryColor:   "#2563eb",
    welcomeMessage: "Salom! Sizga qanday yordam bera olaman?"
  });
</script>
No secrets are embedded in the widget. tenantSlug is a public identifier — the backend enforces all access control.

Configuration Options

Option Type Required Description
tenantSlug string yes Sent as X-Tenant-Slug header on every request
apiBaseUrl string yes Base URL of the FastAPI server (no trailing slash)
title string — Chat window header title
primaryColor hex string — Main brand color (button, header, user bubbles)
welcomeMessage string — First bot message shown when chat opens
leadCaptureMessage string — Prompt shown in the lead capture form
placeholder string — Text input placeholder

API Endpoints Used

MethodPathPurpose
POST /api/chat/message Send message, receive AI reply
POST /api/leads Submit contact form (name + phone)
Session ID is stored in localStorage under sx_session_{tenantSlug} and reused across page reloads.
